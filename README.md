# E-Commerce Frontend Design

This project is a pixel-perfect, desktop-only e-commerce website based on Figma designs. It was developed as part of a frontend development project.

## Project Structure
- `index.html`: Home page
- `products.html`: Product listing page
- `product-details.html`: Product details page
- `css/`: Stylesheets (Vanilla CSS)
- `js/`: JavaScript files for interactivity
- `assets/`: Images and other static resources

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- JavaScript

## Setup
To run the project, simply open any of the HTML files in your preferred web browser. No complex build tools or local servers are required, though a local development server like Live Server (VS Code extension) is recommended for the best experience.

## Features
- Global layout including Header and Footer.
- Desktop-first design approach.
- Interactive product details page with basic JavaScript functionality.
